# Welcome to Whisker Social

Whisker Media is an all-new social media site, developed by Scratchers from all around the globe.

> [!IMPORTANT]  
> We are still in the early stages of development, so many features will not be working yet

## How to Contribute?

If you want to contribute to Whisker Media you can make a Pull Request adding your ideas, or you can make in issue or discussion about why you think you would be a good developer & contributer to Whisker.
