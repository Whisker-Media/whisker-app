### Welcome To Whisker Media!

Whisker Media is a new social media site developed by Scratchers from all around the globe.

Whisker is still a Work In Progress so don't expect the site to work now.

### How to Contribute?

If you want to contribute to Whisker Media you can make a Pull Request adding your ideas, or you can make in issue or discussion about why you think you would be a good contributer to Whisker.
